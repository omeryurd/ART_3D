Detailed Description

This class handles wand tracking for the framework.

This class derives from TrackingBody and implements the methods needed to perform wand tracking operations. While similar to the Head class, the Wand class must also handles joystick and button presses.

Wand Constructor.

double MTF::Wand::GetJoystickHorizontal ( )

Returns the value of the horizontal joystick.

The values for the joystick will range from -1.0 to 1.0. It is typical to assume that a certain range (such as -.15 to .15) will be in the "dead zone" and not considered when using this value. The value returned from here does not account for this, and is left up to the calling method to handle any processing done to the joystick value.

Returns:
Returns the horizontal component of the joysticks as a double
