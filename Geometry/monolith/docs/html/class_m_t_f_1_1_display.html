Detailed Description

This class contains physical information about the display device.

This class stores the corner locations of a display device. These locations give us the width and height of the screen, as well as the up, right, and out vectors of the screen. These locations and vectors are used by the Camera class to create the view and projection matrices for tracking and stereo rendering.
