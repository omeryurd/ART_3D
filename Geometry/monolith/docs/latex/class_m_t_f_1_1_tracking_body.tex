\-This class contains a pure virtual interface for tracking bodies. 

\-Any tracking devices used in the framework must implement the \doxyref{\-Tracking\-Body}{p.}{class_m_t_f_1_1_tracking_body} interface. \-This interface defines a few key methods that all tracking devices share in common.
