\subsection{\-Detailed \-Description}
\-This class handles wand tracking for the framework. 

\-This class derives from \doxyref{\-Tracking\-Body}{p.}{class_m_t_f_1_1_tracking_body} and implements the methods needed to perform wand tracking operations. \-While similar to the \doxyref{\-Head}{p.}{class_m_t_f_1_1_head} class, the \doxyref{\-Wand}{p.}{class_m_t_f_1_1_wand} class must also handles joystick and button presses.
