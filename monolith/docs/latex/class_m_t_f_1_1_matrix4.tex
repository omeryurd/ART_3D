\subsection{\-Detailed \-Description}
\-This class provides for a 4 by 4 matrix and its associated operations. 

\-This class stores a 4x4 double matrix. \-Common functions, such as matrix addition and multiplication is provided through overload operators. \-Less common functions such as matrix inversion and multiplying a vector by a matrix are also included.
