\subsection{\-Detailed \-Description}
\-This class is the primary interface with the \doxyref{\-Monolith}{p.}{class_m_t_f_1_1_monolith} \-Tracking \-Framework. 

\-This class provides methods to get the current \doxyref{\-Camera}{p.}{class_m_t_f_1_1_camera}, \doxyref{\-Head}{p.}{class_m_t_f_1_1_head}, and \doxyref{\-Wand}{p.}{class_m_t_f_1_1_wand} objects. \-It is also responsible for starting the tracker updates and turning them off.
